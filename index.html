Privacy Policy

Tab Group Booker

Effective Date: August 30, 2025

This Privacy Policy explains how the Tab Group Booker browser extension handles data. The extension is designed to operate locally within your browser and does not collect or transmit personal or sensitive user data to external servers.

What the extension does

  • Detects Chrome tab groups and their member tabs within your browser.
  • Creates bookmark folders on your device (Bookmarks Bar) and saves links of tabs that belong to a selected tab group.
  • Adds a right-click (context menu) entry to let you bookmark a tab group quickly.

Data collection and use

  • No external data collection: The extension does not send any browsing information, bookmarks, tab titles, URLs, or other data to any remote server or third party.
  • Local-only operations: All functionality (reading tabs, detecting groups, creating bookmarks) runs locally in the browser using Chrome's extension APIs.
  • No analytics or tracking: The extension does not include analytics, crash reporting, or telemetry services.

Permissions used

The extension requests the following permissions so it can provide its functionality. These are used only locally within your browser:

  • tabs — to read open tabs and their URLs/titles so they can be bookmarked.
  • tabGroups — to detect and identify tab groups for bookmarking.
  • bookmarks — to create bookmark folders and add bookmarks to them.
  • contextMenus — to add a right-click menu item that triggers the bookmark action.

Remote code

This extension does not load or execute remote code. All JavaScript files are included with the extension package and run locally. If remote code execution were introduced in future versions, it would be clearly disclosed in the extension's listing and privacy policy.

Third parties

The extension does not share data with third parties. There are no third-party SDKs, analytics, or ad networks embedded.

Data retention

Since no data is collected or transmitted by the extension, there is nothing retained by the developer. Bookmarks created by the extension remain in the browser's bookmark storage (managed by you and Chrome) and are not accessible to the extension author.

User controls

  • You can remove or disable the extension at any time via chrome://extensions (or your browser's extension manager).
  • Bookmarks created by the extension are ordinary browser bookmarks — you can edit or delete them in Chrome's Bookmark Manager.
